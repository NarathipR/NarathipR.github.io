Autonomous Pesticide Spraying Robot

Description

This project, developed collaboratively by DASLAB at the University of Illinois and Iowa State University, is funded by NSF through the COALESCE initiative[1]. It focuses on creating autonomous robots for precision pesticide spraying in agriculture, integrating multiple components and advanced models to optimize pesticide application. My primary role is to incorporate all software components and design visual servoing systems to enable the robot to perform its pesticide spraying tasks efficiently and accurately.

System Overview

  1. Autonomous Navigation
  2. Bug Detection and Classification
  3. Robot Arm Manipulation

Brief Demo

This demo showcases a robot operating autonomously within a row of crops. All components work seamlessly together, utilizing an autonomous navigation model from [2].
