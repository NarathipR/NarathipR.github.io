1) Forward Kinematics & Inverse Kinematics

Implementation

The system will receive joint angles and return the corresponding end-effector position (forward kinematics), as well as accept a desired position and use the robot's geometry to solve for the required joint angles (inverse kinematics).

2) Hand-Eye Coordination

Implementation

The task involves implementing hand-eye coordination for a robotic system using an overhead camera to view the workspace. The process consists of the following steps:

  • Camera Calibration: Calibrate the top-mounted camera to the robot arm's coordinate system, establishing a precise spatial relationship between the camera's view and the robot's workspace.
  • Image Processing: Implement image filtering techniques to detect and locate colored boxes within the camera's field of view.
  • Hand-Eye Coordination: Utilize the calibrated system and image processing results to enable the robot arm to accurately pick and place the detected boxes within the workspace.
